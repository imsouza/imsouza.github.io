# Portfolio Redesign Design

## Goal

Redesign Mateus Almeida's personal portfolio as a modern, premium Jekyll site while preserving the current minimalist black, gray, white, and blue aesthetic and all existing pages.

## Requirements

- Keep the existing Jekyll structure, pages, posts, notes, tags, projects, and routes.
- Improve visual design across all pages, not only the home page.
- Add a prominent CV download button pointing to `/assets/cv/CV-2.pdf`.
- Improve the bio with Data Science, Artificial Intelligence, Machine Learning, Deep Learning, automation, and Opt Labs context.
- Add a professional career trajectory section with company, role, period, modality/location, description, and company marks.
- Add research projects for reinforcement learning in smart grids and causality/time-series machine learning.
- Refresh skills into professional groups based on the supplied LinkedIn competencies.
- Preserve the established palette and technical/minimal editorial tone.

## Design Direction

The redesign keeps the current monochrome identity and monospace/sans technical feel, but adds stronger spacing, card surfaces, timeline structure, clearer buttons, and responsive page shells. The site should read as a professional AI/data portfolio, not a marketing landing page.

## Pages

- `/`: bio, CV action, social links, career timeline, research, skills summary.
- `/projetos/`: modern project cards with image, category, description, keywords, and external action.
- `/posts/`, post pages, `/notas/`, notes, `/tags/`, `/skills`, `/publicacoes/`, `404`: unified typography, spacing, lists, tags, and empty/error states.
